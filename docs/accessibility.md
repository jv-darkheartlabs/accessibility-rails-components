# Accessibility Guidelines

Goals:
- Meet or exceed WCAG 2.1 AA
- Support keyboard-only navigation
- Support screen readers (VoiceOver, NVDA/JAWS where possible)

Checklist:
- Logical heading order
- Visible focus states
- Label form controls
- Announce dynamic changes (ARIA live regions) when required

Tools:
- axe-core / Axe CLI for quick audits
